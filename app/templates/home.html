What is UbSRD?

UbSRD is a resource created to help classify and search ubiquitin-like fold containing structures. UbSRD also provides relational database interface for simplified analysis of the structural features of ubiqutin-like molecules and their interaction partners. Structures in UbSRD are grouped by ubiquitin-like fold type and mode of protein-protein interaction, then we used the scientific benchmarking tools in Rosetta3 to catalouge various features of this set 1. The purpose of this website is to provide a frontend for simple browsing of the UbSRD database. To interact with the database online, start with the structure browsing page. To do more complex structural analysis, download the database and check out our example analysis page. If you are interested in using UbSRD or have ideas for improvements please contact joseph_harrison"at"med.unc.edu.

If you find the data contained in UbSRD useful, please reference our article.
